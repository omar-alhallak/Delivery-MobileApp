🌳 Branching Strategy
    

طريقة العمل على البيئة :
1_أول شي بفوت بعمل فيوتشر من Develop للشي الي بدي ضيفو ل Develop .
2_بعمل بول من Develop لل فيوتشر الي ضفتا.
3_بلش بعدل أو بضيف أي شي بدي ياه أو أي شي عم ساويه الخ.
4_خلصت شغل شلون بحفظ :
أ_أول شي بعمل pull لل Develop  شان اذا في أي شي تعدل يكون تحدث عندي.
ب_بفوت بعمل ميرش للفيوتشر ع Develop خلصت.

5_بحذف الفيوتشر دغري بعد ما عملتلها ميرش.
ملاحظة : اذا عملت ميرش عالفيوتشر لو نسيت شي لا تكتبو بالفيوتشر حذفها وعميل فيوتشر جديدة عدل عليها.
6_بعمل pull لل Develop.
وهيك خالص

ملاحظة : dependence وال migration ممنوع حدا يعدل أو يضيف شي عليهم .
وممنوع حدا يعمل ميرش عال main ال main محد بيقرب عليها.
وممنوع تحذف سطر أو تعدل أي شي عال Develop أو عال main .
أي شي من الي فوق اذا صار بيضرب المشروع بالكامل.

في أي استفسار سئلني أو اذا في أي سؤال على البيئة مافهمت شي ما عرفت تساوي شي الخ... بعد الفحص.
